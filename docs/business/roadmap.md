## مراحل التنفيذ التقني (Engineering Phases)
| Phase | النطاق | الحالة |
|-------|--------|--------|
| 0 | الأساس: Monorepo، Backend foundation، Design system | ✅ |
| 1 | الهوية والأمان: Auth / OTP / RBAC / Documents | 🔄 |
| 2 | النقل: Universities / Areas / PickupPoints / Routes / Subscriptions / Trips / Tracking | ⏳ |
| 3 | الدفع: Payment Requests / CliQ / GPT Vision / Financial Transparency | ⏳ |
| 4 | الخدمات: Parcels / Lost & Found / Rewards | ⏳ |
| 5 | الذكاء الاصطناعي: Rafeeq Assistant / Support / Complaint / Safety | ⏳ |

| 7 | الإطلاق: SOS / Notifications / Deployment / Hardening | ⏳ |

## نقاط القوة التنافسية
- أول منصة تجمع النقل والخدمات الطلابية.
- نظام أمان متعدد الطبقات.
- دعم ذكي مع GPT.
- إرسال أغراض عبر شبكة الرحلات.
- نظام مفقودات ذكي.
- تحليلات ذكاء اصطناعي.
- قابلية توسع عالية + إمكانية التعاقد مع الجامعات.
- نموذج تشغيلي قابل للاستثمار.
- بنية تسمح بإضافة خدمات جديدة دون إعادة بناء النظام.
