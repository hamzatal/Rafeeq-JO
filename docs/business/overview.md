# رفيق | نظرة عمل المشروع (Business Overview)

## الهوية
- **الاسم:** رفيق (Rafeeq)
- **التصنيف:** منصة نقل وخدمات طلابية ذكية مدعومة بالذكاء الاصطناعي.
- **الرؤية:** أن تصبح رفيق المنصة الأولى للنقل والخدمات الجامعية في الأردن، ثم التوسع إقليمياً.
- **الرسالة:** تجربة آمنة وسهلة وذكية للطلاب والكباتن والجامعات عبر نظام موحّد.

## فلسفة المنصة
رفيق ليست تطبيق حجز، ولا تطبيق توصيل، ولا تطبيق جامعي فقط. رفيق هي:
- شبكة نقل جامعية
- منصة خدمات طلابية
- شبكة تبادل وإرسال أغراض
- نظام أمان ومتابعة
- مساعد ذكي يعتمد على GPT
- منصة تشغيل وإدارة للجامعات مستقبلاً

## الفئات المستهدفة
| الفئة | الدور |
|-------|-------|
| الطالب | المستخدم الأساسي |
| الكابتن | السائق المعتمد |
| الإدارة | إدارة المنصة |
| الدعم الفني | فريق المساندة |
| الجامعات (مستقبلاً) | جهات متعاقدة |
| الشركات (مستقبلاً) | النقل المؤسسي |

## الخدمات الأساسية
1. **النقل الجامعي** — الاشتراكات، الرحلات اليومية، التتبّع المباشر، التقييمات، الإشعارات.
2. **إرسال الأغراض** — كتب/مفاتيح/وثائق/شواحن/أدوات قانونية، من جامعة لجامعة أو نقطة لنقطة.
3. **المفقودات (Lost & Found)** — الإبلاغ + مطابقة ذكية بالـ GPT.
4. **التبادل الطلابي (Smart Campus Exchange)** — تبادل كتب/ملازم/مشاريع/أدوات.

## محرّك الذكاء الاصطناعي
- **Rafeeq AI** — مساعد داخل التطبيق (أسئلة، اشتراكات، توجيه).
- **AI Support Agent** — تحليل التذاكر، تصنيف المشكلة، اقتراح الحلول.
- **AI Complaint Analyzer** — تحليل الشكاوى وتحديد الخطورة وإنشاء تقرير.
- **AI Safety Monitor** — مراقبة التقييمات/الشكاوى/التأخير/الإلغاءات واكتشاف الأنماط الخطرة.
- **AI Route Intelligence** — أفضل نقطة تجمع/مسار/كابتن/وقت انطلاق.
- **AI Lost & Found Matching** — ربط المفقودات بالموجودات.
- **AI Analytics** — تقارير ذكية للإدارة.

## نموذج الدفع
- المرحلة الحالية: **CliQ Jordan**.
- لا يحوّل الطالب عشوائياً — يُنشأ **Payment Request** برقم فريد (مثال: `RFQ-2026-83472`) ومبلغ ووقت انتهاء.
- التحقق عبر **GPT Vision** (تحليل صورة التحويل: المبلغ/الوقت/رقم الطلب).
- **مركز الشفافية المالية:** كل عملية لها رقم/حالة/وقت دفع/وقت اعتماد/سجل كامل.
- **ضمان رفيق:** إذا تأخر الاعتماد تُفتح تذكرة دعم تلقائياً.
- مستقبلاً: Request To Pay → بوابة دفع → محفظة رفيق.
