# سياسة الطرود الممنوعة — رفيق (Rafeeq)

عند إرسال طرد عبر رفيق، تُقرّ بأن محتواه لا يشمل أياً ممّا يلي:

1. المواد غير القانونية أو المخدّرات أو مشتقّاتها.
2. الأسلحة والذخائر والمواد القابلة للانفجار أو الاشتعال.
3. المواد الكيميائية/السامة الخطرة.
4. النقود والمجوهرات والمعادن الثمينة (غير مشمولة بأي ضمان).
5. الكائنات الحيّة.
6. المواد القابلة للتلف بسرعة دون تغليف مناسب.
7. أي مادة تنتهك القوانين الأردنية.

**الإقرار**: المُرسِل مسؤول قانونياً عن المحتوى. يحق لرفيق وللكابتن رفض/إيقاف أي طرد مشبوه وإبلاغ الجهات المختصّة. التسليم/الاستلام يتمّ بتأكيد **OTP مزدوج** مع توثيق سلسلة العهدة.
