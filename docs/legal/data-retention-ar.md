# سياسة الاحتفاظ بالبيانات — رفيق (Rafeeq)

| نوع البيانات | مدة الاحتفاظ |
|---|---|
| بيانات الحساب | طوال نشاط الحساب + حتى الحذف |
| سجلّات الموقع الحيّة (trip_tracking) | تُجمّع/تُقلّص بعد اكتمال الرحلة؛ يُحتفظ بملخّص للأمان |
| سجلّات OTP والإلغاء | حتى 12 شهراً (أدلة مكافحة احتيال/نزاعات) |
| المدفوعات وإشعارات التحويل | وفق المتطلبات المحاسبية/القانونية (حتى 5 سنوات) |
| التذاكر والشكاوى | حتى 24 شهراً من الإغلاق |
| سجلّات التدقيق (audit_logs) | غير قابلة للتعديل؛ تُحتفظ للمراجعة الأمنية |

عند حذف الحساب: تُزال البيانات الشخصية القابلة للتعريف عدا ما يفرض القانون أو التحقيقات الجارية الاحتفاظ به (تُجهَّل/تُخفى الهوية حيثما أمكن).
