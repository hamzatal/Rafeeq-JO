## المكوّنات الموحّدة (تطبيق الطالب — `src/components/`)
- **Icon** (`Icon.tsx`): غلاف موحّد فوق `@expo/vector-icons` (Feather) — نقطة تبديل واحدة.
- **ui.tsx**: `Card` · `ScreenHeader` · `SectionTitle` · `Badge` (نغمات) · `EmptyState` · `ServiceTile` · `StatCard` · `ListRow` — كلها RTL + ثيم + ظلال.
- **Button / Input / Banner / Screen**: المكوّنات الأساسية السابقة (ترث الثيم الجديد تلقائياً).

## التنقّل
- تطبيق الطالب: **Bottom Tabs** (الرئيسية · رحلاتي · المحفظة · الإشعارات · حسابي) بأيقونات، والشاشات الثانوية مخفيّة من الشريط وتُفتح بالتنقّل.

## الشاشات المُعاد بناؤها
- **الرئيسية**: ترويسة بالأفاتار + جرس إشعارات (بنقطة غير المقروء) + **CTA «اطلب رحلة»** + بطاقة رصيد المحفظة + شبكة خدمات بأيقونات.

## التالي (دفعات قادمة)
- إعادة بناء بقية شاشات الطالب (الاشتراكات/الرحلات/المحفظة/الدفع/الطرود...) فوق المكوّنات الجديدة.
- تطبيق نفس النظام على تطبيق الكابتن ولوحة الإدارة.
- حالات تحميل (Skeleton) + انتقالات + onboarding.

## خلفية السبلاش وخريطة التتبّع (RFQ-216)

### خلفية الخريطة على السبلاش — `MapBackdrop`
- مكوّن SVG (`react-native-svg`) في `*/src/components/MapBackdrop.tsx` (طالب + كابتن).
- يرسم **خريطة طرق أردنية مبسّطة**: شبكة طرق منحنية + طرق سريعة أعرض + عُقد مدن + **مسار رحلة مميّز** بلون ذهبي متقطّع.
- قابل للثيم عبر props: `roadColor`, `routeColor`, `nodeColor`, `opacity`.
- الاستخدام: خلفية مطلقة (`absoluteFill`) خلف شعار السبلاش + طبقة vignette خفيفة لإبقاء الشعار واضحاً.
- ملاحظة: طُلبت صورة `Amman Map.png` لكنها لم تكن موجودة فعلياً في الريبو، فاستُبدلت بهذا البديل المتجه (أخف، متجاوب، بلا أصل ثنائي). لاستخدام صورة حقيقية لاحقاً: ضعها في `assets/` واستبدل `MapBackdrop` بـ `<ImageBackground>`.

### تصميم خريطة التتبّع الحيّة — `LiveMap`
- يعمل بمحرّكين: **Google Maps JS API الرسمي** (عند توفّر المفتاح) و**Leaflet/OSM** (بديل مجاني)، بنفس واجهة التحديث الحيّ.
- **الدبابيس (Markers) العصرية**:
  - نقطة الالتقاط: دبوس **teardrop** بظل ناعم + قلب أبيض + رقم/أيقونة.
  - الكابتن: **شارة دائرية** بأيقونة سيارة SVG + حلقة نبض (pulse) + حركة انزلاق ناعمة بين التحديثات.
  - الوجهة (★) والانطلاق (⌂): دبابيس teardrop مميّزة بالألوان.
- **خط المسار**: casing أبيض سفلي + خط ملوّن علوي بأطراف دائرية (مظهر تطبيقات الخرائط الاحترافية).
- الألوان من الثيم: الكابتن=primary، الالتقاط=أزرق، الوجهة=ذهبي، الانطلاق=أخضر نجاح.
