### الجودة (أوامر سريعة)
- الاختبارات: `cd backend && ./vendor/bin/phpunit`
- التحليل الساكن: `cd backend && vendor/bin/phpstan analyse` · الستايل: `vendor/bin/pint`
- فحص أنواع الفرونت: لكل تطبيق `npm run typecheck`

> ملاحظة: كل الوثائق/الخطط/الهويات القديمة (7 تصاميم + خطط متكرّرة + سجلّات مهجورة) **حُذفت وأُعيدت الهيكلة** في هذه البنية لتفادي التشتّت.
