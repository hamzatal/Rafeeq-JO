## 🧪 الجودة (Quality)

- **الاختبارات:** `cd backend && php artisan test`.
- **التحليل الساكن:** `cd backend && vendor/bin/phpstan analyse` (Larastan، إعداد في `phpstan.neon`).
- **فحص الأنواع (الفرونت):** لكل تطبيق `npx tsc --noEmit`.
- **الستايل:** `cd backend && vendor/bin/pint`.
